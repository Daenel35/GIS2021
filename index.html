The two layers exposed at the following are the result of the implementation of a random forest model with two different approaches (1000 sampling points and 5000 sampling points) in order to determine which are the areas were is more likely that a landslides occurs. The study region was Tirano.


Brief about the processing implemented

In order to predict the landslide's susceptibility zones of the assigned region was necessary to carry out a set of steps to process a certain group of inputs. Those steps have covered mainly the following aspects:

  • implementation of basic fixing operations on the input layers like resampling and reclassification of their data
  • transformation from vector data model to raster one
  • extraction of information by implementing a sampling technique over the different input layers
This preprocessing stage was the foundation for the implementation of the random forest model.
